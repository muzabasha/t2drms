# TRL-2: AI-Enabled Personalized Type-2 Diabetic Management System

## 🏥 Research Demonstration Application

**Technology Readiness Level:** TRL-2 (Technology Concept and Application Formulated)

### Project Overview

This is an interactive research demonstration of an AI-powered diabetes management system that integrates:
- **OCR Module**: Medical report digitization and entity extraction (94.5% accuracy)
- **Federated Learning**: Privacy-preserving multi-hospital AI training (ε=0.5 differential privacy)
- **Agentic AI**: 5 specialized AI agents for holistic patient care
- **Food Intelligence**: Image-based food recognition and glycemic impact analysis (91.2% accuracy)
- **Personalized Diet Planning**: Real-time meal optimization based on patient profiles

### 🚀 Quick Start

1. **Open the demonstration:**
   ```
   Open demo.html in a web browser
   ```

2. **Navigate through 6 interactive steps:**
   - Step 1: Select a patient
   - Step 2: OCR extraction demo
   - Step 3: Federated learning visualization
   - Step 4: Food intelligence analysis
   - Step 5: Personalized diet plan
   - Step 6: Complete pipeline summary
